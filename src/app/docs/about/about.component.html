关于

公司项目前端主要基于Angular ng-zorro框架开发 https://ng.ant.design/
本项目的主要目的是针对公司现在项目需求，提供ng-zorro框架中未有提供的前端组件

组件基于现代浏览器开发，兼容IE11以上浏览器
所有组件均以"ecp-"开头

目录说明


├── src
│   ├── app
│   │   ├── components                          # 自定义组件文件夹
│   │   │  
│   │   ├── demo                                # 演示页面文件
│   │   ├── docs                                # 组件说明文件夹
│   │   │   ├── about                           # 关于
│   │   │   └── components                      # 组件说明文件
│   │   ├── app.component.ts                    # 根组件
│   │   └── app.module.ts                       # 根模块
│   ├── assets                                  # 本地静态资源
│   ├── environments                            # 环境变量配置
│   ├── styles                                  # 样式目录
└── └── style.less                              # 样式引导入口
